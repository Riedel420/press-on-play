# Press-On Play™

A modern web-based nail design studio built with React, TypeScript, and Three.js.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Three.js
- Express
- Drizzle
- Radix UI
- Tailwind CSS

## Deployment

1. Build the project:
   ```bash
   npm run build
   ```

2. Test the production build:
   ```bash
   npm run preview
   ```

3. Deploy to your hosting platform of choice:
   - Push to `main` for GitHub Pages automatic deployment
   - Or run `npm run deploy` for manual deployment
